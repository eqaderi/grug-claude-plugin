# Changelog

## 0.1.0 — 2026-06-16

Initial release.

- `grug-engineering` skill: simplicity-first engineering judgment, auto-triggered on review, planning, refactoring, and complexity questions
- `/grug-review` command: read-only complexity review of code changes
- `/grug-plan` command: plan a task using smallest-change-first principles
- `/grug-refactor` command: safe step-by-step refactoring with Chesterton's Fence analysis
- `global-rules.md`: optional always-on instructions for `~/.claude/CLAUDE.md`
- `repo-snippets/CLAUDE.grug-snippet.md`: optional per-repo reminder snippet
